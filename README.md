# LeetCode Solutions

Репозиторий с решениями алгоритмических задач с LeetCode на Java.

Я использую этот репозиторий для тренировки алгоритмического мышления, работы со структурами данных и подготовки к техническим собеседованиям.

## Почему Java?

Java используется здесь как язык для алгоритмической практики.  
Он строгий, типизированный и удобен для отработки структур данных, анализа сложности и чистой логики решения.

Моё основное направление прикладной разработки — JavaScript, TypeScript и frontend-разработка. Поэтому практические web-проекты я развиваю на JavaScript/TypeScript, а этот репозиторий использую именно для алгоритмов.
